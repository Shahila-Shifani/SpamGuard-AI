# SpamGuard-AI
An NLP-based SMS spam classification system using machine learning to detect spam and legitimate messages with text preprocessing, TF-IDF feature extraction, and predictive modeling.

## NLP-Based SMS Spam Classification System

SpamGuard AI is a machine learning-based application that detects whether an SMS message is spam or legitimate using Natural Language Processing techniques.
